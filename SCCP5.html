O QUE É O JAVA SCRIPT ?

É uma linguagem de programação usada principalmente para tornar páginas web interativas. Ela roda diretamente no navegador do usuário e permite que os sites respondam a ações como cliques, envios de formulários e movimentos do mouse. Com JavaScript, você pode atualizar partes de uma página sem recarregá-la toda, validar dados de formulários antes do envio, criar animações e muito mais. Originalmente criada apenas para o lado do cliente, hoje também é usada no servidor (com Node.js), em aplicativos móveis e até em programas de desktop. É a linguagem que dá "vida" às páginas web, trabalhando junto com HTML (que define a estrutura) e CSS (que cuida do estilo visual).
